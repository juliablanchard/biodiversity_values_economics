This directory contains BII v2.1.1 data, temporally limited. We include resolution rasters with a spatial resolution of approximately 10km at the equator for the years 2000, 2005, 2010, 2015, and 2020. BII rasters are percentage values between 0 and 100, indicating the intactness of the ecosystem: 0 meaning completely degraded and 100 being completely intact.

Metadata:
File type: Geotiff (.tif file extension, with auxiliary files holding the spatial metadata)
Geographic coordinate system: World Geodetic System 1984 (WGS84, EPSG: 4326)
Spatial resolution: 5 arc minutes (approximately 10km at the equator)
Spatial extent: global (xmin = -180, xmax = 180, ymin = -90, ymax = 90)
